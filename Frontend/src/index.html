Welcome to Chunking site!

This site allows you to split larger files into chunks. it is easy, fast and very convinient to use. You can chunk files of various types such as Json, CSV_ files and lots more. When sending a large file to friends or uploading to a website, it is preferable to split large files into chunks especially in Windows as it is much easier to send and upload smaller files than the entire file. This site also give the user the opportunity of saving files after chunking, uploading files, downloading files. The user can also select the number of parts and sizes of parts the files will be chunked into.
